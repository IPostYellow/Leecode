# 图

1. dfs

   ```python
   def dfs(cur,graph,visited):
       '''
       cur:当前结点
       graph:图的邻接表(如果是邻接矩阵需要简单修改下代码)
       visited:用来记录已经访问过的结点
       '''
       if visited[cur]==1:
           return 访问过就不需要再访问了
       visited[cur]=1
       for i in graph[cur]: #访问这个结点连接的下一个结点，不停递归
           dfs(i,graph,visited)
       return 从cur开始的dfs遍历结束
   ```

2. bfs

   ```python
   from queue import Queue
   def bfs(cur,graph,visited):
       '''
       cur:当前结点
       graph:图的邻接表(如果是邻接矩阵需要简单修改下代码)
       visited:用来记录已经访问过的结点
       '''
       if visited[cur]==1:
   		return 访问过了
       que=Queue()
       que.put(cur)
       while not que.empty():
           tmp=que.get()
           visited[tmp]=1
           for i in graph[tmp]:
               if visited[i]!=1:
                   que.put(i)
       return 从cur开始的bfs遍历结束
   ```
